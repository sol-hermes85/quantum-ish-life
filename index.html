Drag to paint; desktop mouse users get a filled cell preview first and the cursor changes for erase and pan modes. Bloom paints a soft probability cluster, while Time Echo leaves fading traces of recent cells. The Tool stat shows the active paint/erase mode and Dot/Bloom brush. Alt-drag temporarily flips paint and erase, and the hover preview follows that temporary tool before you draw. Shift-drag, middle-drag, or right-drag pans the grid. Double-click or double-tap resets the view. Press F to frame the current live cells, or use the frame button to centre them; selected preset patterns auto-frame when loaded. On small screens, Play folds the tuning controls away so the grid stays open. Clear asks for one quick confirmation and briefly marks the button before wiping the grid. Hidden tabs pause automatically to save battery. The trend stat shows whether the live population is rising, falling, or steady, with a subtle colour cue. Brief feedback appears over the grid when the main mode, brush mode, pinch zoom, button zoom, or grid content changes, including paint/erase stroke counts and the generation reached when pausing; longer messages linger briefly so they are readable.
